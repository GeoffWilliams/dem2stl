# dem2stl

Convert digital elevation maps (DEM) to 3d printer stereolithography format (STL)

## Libraries

```
pip install image
pip install numpy
```

## Example

```
./dem2stl.py --input-file ~/Downloads/20210709_172012.jpg --model-depth 10 --output-file breathe.stl --resample-size 6 --linear-ratio 40
```
